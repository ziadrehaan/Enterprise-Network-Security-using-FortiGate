# 🛡 Enterprise Network Security using FortiGate

A complete enterprise security infrastructure built using **FortiGate Firewall** as the final project for the **Digital Egypt Pioneers Initiative (DEPI) – Fortinet Cybersecurity Track**.

This project demonstrates how to design, configure, monitor, and validate a multi-layered defense architecture using Fortinet security technologies.

---

# 🚀 Project Overview

The project implements a complete enterprise security solution capable of:

✅ Stateful Firewall Policies

✅ Application Control

✅ Web Filtering

✅ Antivirus Protection

✅ SSL Inspection

✅ Threat Weight Configuration

✅ Logging & Security Events

✅ FortiView Monitoring

✅ Real-Time Traffic Analysis

# 🔥 Security Features

### Firewall Policies

- Flow-Based Inspection
- NAT Configuration
- Security Profile Integration
- Session Logging

### Application Control

- Block Facebook
- Block Instagram
- Block YouTube
- Detect Unknown Applications
- Log Allowed & Blocked Applications

### Web Filtering

- Category-Based Filtering
- URL Filtering
- FortiGuard Ratings
- Safe Browsing Enforcement

### Antivirus

- Flow-Based Malware Inspection
- HTTP / HTTPS Protection
- FTP Protection
- SMB & CIFS Inspection
- EICAR Malware Detection

### Logging & Monitoring

- Forward Traffic Logs
- Security Event Logs
- Threat Weight
- Local Logging

### FortiView

- Live Applications
- Sources & Destinations
- Sessions
- Threats
- Traffic Analytics
